# International Data Security User Group - Session Slides

Welcome to the official repository for the International Data Security User Group! 

## About This Repository

This repository serves as a central archive for all presentation slides and materials from our user group sessions. Here you'll find resources from our expert speakers who share their knowledge and practical insights on data security topics, with a focus on Microsoft Purview and related technologies.

## What You'll Find Here

- **Session slides** from all past meetings
- **Presentation materials** shared by our guest speakers
- **Additional resources** referenced during sessions

Each folder is organized by session date and topic for easy navigation.

## About the User Group

The International Data Security User Group brings together data security professionals from around the world to share knowledge, experiences, and best practices. Our sessions feature industry experts who provide both strategic context and practical guidance on implementing effective data security solutions.

### Session Format
- Monthly virtual meetings
- Expert-led presentations
- Interactive discussions and knowledge sharing
- Practical, real-world insights you can apply immediately
